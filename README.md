# InterviewAI-Your-Personal-AI-Powered-Placement-Coach
InterviewAI is a premium placement prep coach built with Streamlit. It leverages IBM Granite 4 for smart resume parsing, ATS compatibility matching, mock interviews, tailored coding sandbox challenges, and timed technical/aptitude MCQ quizzes, with optional voice-to-text response tracking powered by Groq Whisper STT. 🎤🚀
